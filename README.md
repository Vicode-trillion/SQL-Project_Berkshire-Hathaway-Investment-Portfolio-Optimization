# SQL-Project_Berkshire-Hathaway-Investment-Portfolio-Optimization

A data-driven portfolio analysis project simulating a real-world client engagement for Berkshire Hathaway — a $270B institutional fund. Using SQL for quantitative modeling and financial analysis, this project identifies a critical concentration risk and delivers specific, evidence-backed rebalancing recommendations.

⛰️ The Business Problem: 

Berkshire Hathaway's top 5 holdings (AAPL, BAC, KO, AXP, CVX) were significantly misaligned — the portfolio had its largest position in its worst-performing stock and its smallest position in its best-performing stock. The goal was to diagnose this inefficiency and recommend a data-supported rebalancing strategy.

 🎯 What This Project Does:

Queried a structured financial database to pull holding values, returns, and risk metrics across five stocks (2021–2023)
Calculated Sharpe Ratios (risk-adjusted return) for each position to surface where capital was being under- or over-allocated
Built a side-by-side comparison of the current vs. optimized portfolio
Translated findings into four concrete rebalancing recommendations with projected financial impact

💡 The Recommendation:

Four targeted weight adjustments — cutting AAPL by half, increasing CVX, AXP, and KO — projected to deliver:
Return: 20.16% → 23.61% (+3.45%)
Portfolio Risk: 29.03% → 28.16% (−0.87%)
Sharpe Ratio: 0.556 → 0.696 (+25%)
Dollar Impact: +$9.3B in additional annual returns on $270B AUM

-SQL querying — pulling and aggregating financial data from a structured database
- Quantitative analysis — Sharpe Ratio calculation, annualized returns, volatility modeling
- Business storytelling — translating raw numbers into a clear, executive-ready recommendation
- Presentation under pressure — project delivered under a strict time constraint to simulate a real client deadline
